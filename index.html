Cloud sync

Sync your tasks across devices through a private GitHub Gist. You need a token with gist scope — create one at github.com/settings/tokens.

Your token is stored only in this browser and sent only to GitHub. Edits push automatically; the app pulls the cloud copy when you open or return to it. Latest save wins. To add a second device: copy the Gist ID here, then on the other device paste the same token + this ID and tap Connect.
